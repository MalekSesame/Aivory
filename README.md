# Mini RAG – Question/Réponse sur PDF

## Objectif
Développer une application RAG permettant d’interroger des documents PDF
via une interface simple.

## Technologies
- FastAPI
- Streamlit
- Sentence-Transformers
- ChromaDB
- OpenAI GPT
- Docker

## Lancement
docker-compose up --build

## Accès
- API : http://localhost:8000/docs
- UI : http://localhost:8501
